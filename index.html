PROJECTS

Election Website

This website houses all 2020 election-related content for The Michigan Daily, including articles, quizzes, and visualizations. I was involved in the creation of quiz framework software, data scraping + visualization, and website development using React. WEBSITE LINK.

Feeder Schools

I led a data-driven series of articles in which The Michigan Daily obtained records on the top 301 schools by number of applications to the University of Michigan for the Fall 2019 freshman class through a public records request. I analyzed demographic and geographic data as well as developed visualizations to publish in the series. ARTICLE LINK.

Miseducation

A collection of summer 2019 articles which dissect systemic tendencies and cultural norms which work in tandem to create racial inequality in America. I worked on front-end website development. WEBSITE | GITHUB

Revenue CBP
